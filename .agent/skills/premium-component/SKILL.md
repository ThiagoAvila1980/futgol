# Premium UI Component Skill

Esta skill deve ser usada sempre que um novo componente de interface for criado ou um existente for refatorado para melhorar sua estética.

## Princípios de Design
1. **Estética Rica**: Use cores vibrantes, modos escuros elegantes e glassmorphism quando apropriado.
2. **Tipografia Moderna**: Utilize fontes como 'Inter' ou 'Outfit'.
3. **Micro-animações**: Adicione hover effects, transições suaves e feedback visual imediato.
4. **Consistência**: Siga a paleta de cores do projeto (veja `tailwind.config.js`).

## Checklist de Implementação
- [ ] O componente usa TailwindCSS de forma eficiente?
- [ ] Possui estados de hover/active?
- [ ] É responsivo (mobile-first)?
- [ ] Segue o padrão de acessibilidade (labels, roles)?
- [ ] O código está modularizado e tipado?

## Como Usar
Para criar um novo componente usando esta skill, você pode me pedir:
"Crie um componente [Nome] usando a skill Premium UI Component."
